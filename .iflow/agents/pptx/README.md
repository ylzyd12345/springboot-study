PPTX Workflow - PowerPoint演示文稿智能处理工作流

一键安装：
在iFlow CLI执行命令（"iflow workflow..."替换为网站提供的安装指令）：
示例：!iflow workflow add "pptx-aBcDef"
安装完成后重启iFlow CLI，输入 `/pptx` 使用。

核心功能：
1. 创建演示文稿
从零创建专业演示，支持自定义设计。
示例：/pptx 创建产品发布演示，5张幻灯片
示例：/pptx 制作年度报告，蓝色主题

2. 文本提取
快速提取演示文稿文本和备注。
示例：/pptx 从report.pptx提取文本
示例：/pptx 读取演讲者备注

3. 使用模板
基于模板快速创建演示文稿。
示例：/pptx 用template.pptx创建演示
示例：/pptx 复制模板第1、3、5张

4. 编辑演示
修改演示文稿内容和布局。
示例：/pptx 修改第2张标题
示例：/pptx 添加销售图表

5. 添加图表
在幻灯片插入可视化元素。
示例：/pptx 添加柱状图
示例：/pptx 创建对比表格
