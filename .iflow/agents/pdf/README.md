Docx Workflow - Word文档智能处理工作流

一键安装：
在iFlow CLI中执行以下命令（"iflow workflow..."替换为网站提供的安装指令）：
示例：!iflow workflow add "pdf-rFQkrA"
安装完成后重启iFlow CLI，在输入行添加 `/pdf` 指令即可使用。

核心功能：
1. 文本提取
提取PDF文本内容，保持原始布局。支持纯文本和扫描型PDF（OCR）。
示例：/pdf 从report.pdf中提取全部文本

2. 表格提取
提取表格数据并导出为Excel、CSV格式，适合处理财务报表和数据统计。
示例：/pdf 提取financial_statement.pdf中的表格并导出为Excel

3. 文档创建
编程创建专业PDF文档，支持文字、表格、图片等元素。完整支持中文字体。
示例：/pdf 创建包含公司介绍和数据表格的宣传册PDF

4. 合并与拆分
合并多个PDF或拆分大型PDF，支持自定义页面顺序。
示例：/pdf 将part1.pdf、part2.pdf、part3.pdf合并为完整文档
示例：/pdf 将报告拆分为多个文件，每10页一个

5. 表单填写
自动填写PDF表单字段，支持可填写和非填写表单。
示例：/pdf 填写申请表，姓名"张三"，日期"2025-10-23"

6. OCR识别
对扫描型PDF进行文字识别，支持中英文混合文档。
示例：/pdf 对扫描文档进行OCR识别并提取文本

7. 水印与加密
添加水印或密码保护，支持自定义位置、透明度。
示例：/pdf 给合同添加"机密"水印
示例：/pdf 给文档添加密码保护

8. 元数据管理
读取和修改PDF元数据（标题、作者、创建时间等）。
示例：/pdf 读取报告的元数据信息
