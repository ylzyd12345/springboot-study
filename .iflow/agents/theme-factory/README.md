Theme Factory - 专业主题工厂智能应用工具

一键安装：
在iFlow CLI执行命令（"iflow workflow..."替换为安装指令）：
示例：!iflow workflow add "theme-factory-aBcDef"
安装后重启iFlow CLI，输入 `/theme-factory` 使用。

支持的文档类型
- PPT演示文稿
- HTML文档
- Word文档
- 其他可视化文档：各类需要专业配色的内容

核心功能:
1. 主题展示
可视化PDF展示10种专业主题配色
示例：`/theme-factory 显示所有主题`

2. 预设主题库
10种精选主题：Ocean Depths（海洋深度）、Sunset Boulevard（日落大道）、Forest Canopy（森林树冠）、Modern Minimalist（现代极简）、Golden Hour（黄金时刻）、Arctic Frost（极地冰霜）、Desert Rose（沙漠玫瑰）、Tech Innovation（科技创新）、Botanical Garden（植物花园）、Midnight Galaxy（午夜星系）

3. 快速应用
一键应用专业配色方案
示例：`/theme-factory 应用Ocean Depths`

4. 自定义主题
根据需求即时创建全新主题
示例：`/theme-factory 创建紫色优雅主题`

与其它工作流搭配使用:
与 /pptx 搭配
场景一：创建带主题的演示文稿
步骤1：/pptx 创建产品介绍PPT
步骤2：/theme-factory 应用Ocean Depths主题

场景二：一步到位创建
/pptx 创建公司年度报告，使用/theme-factory的Sunset Boulevard配色

场景二：与其他文档工具搭配
示例1：/theme-factory 为当前PPT文档应用Modern Minimalist主题
示例2：创建营销报告后，使用 /theme-factory 应用Tech Innovation配色
