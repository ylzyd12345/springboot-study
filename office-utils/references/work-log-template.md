# 工作日志模板

## AI 初始分析模板

```markdown
# [任务名称] 工作记录

## AI 初始分析

**分析时间：** YYYY-MM-DD  
**分析目录：** [目录名]

### 文件清单

| 简称 | 原文件名 | 类型 | 说明 |
|------|----------|------|------|
| 【xxx】 | `文件名.xlsx` | Excel | 简要说明 |
| 【xxx】 | `文件名.docx` | Word | 简要说明 |
| 【xxx】 | `文件名.pdf` | PDF | 简要说明 |

### Excel 文件分析

#### 【简称】`文件名.xlsx`

| Sheet | 角色 | 行数 | 说明 |
|-------|------|------|------|
| Sheet1 | 明细表 | 1234 | 主数据 |

**表头：** 列1, 列2, 列3...

**关键字段：**
| 字段 | 类型 | 说明 |
|------|------|------|
| 机构编号 | 标识 | 主键 |
| 金额 | 金额 | 统计字段 |

### Word/PDF 文件分析

#### 【简称】`文件名.docx`

**结构：** X个段落，Y个表格  
**主要内容：** [内容摘要]

### 任务推断

**任务类型：** [核查任务/统计汇总/数据填报/对账核对]  
**判断依据：** [关键词、文件结构等]

### 建议步骤

1. ...
2. ...

---

## 工作记录

<!-- 后续记录 -->
```

## 日常记录格式

### 日期标题
```markdown
### YYYY年MM月DD日 [事项]
```

### 文件简称
```markdown
【简称】 → `原文件名.xlsx`
```

### 决策记录
```markdown
就 [问题]，决定 [方案]。
```

### 填写规则
```markdown
列X（列名） ← 数据来源
```

### 争议记录
```markdown
**争议N：[标题]**
[内容]
```

### 版本对比
```markdown
| 项目 | 旧版 | 新版 | 说明 |
|------|------|------|------|
```
